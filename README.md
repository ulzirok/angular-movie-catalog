# Проект Movie App
Movie App — приложение для просмотра каталога фильмов с возможностью поиска и просмотра подробной информации о фильме.

Проект выполнен в рамках тестового задания.

## Ссылка на демо: 

## Автор: 
Улзира Пердебаева

## Функциональность: 
- Отображение списка фильмов
- Поиск фильмов по названию
- Просмотр карточки фильма с подробной информацией
- Загрузка данных с внешнего API
- Адаптивная верстка

## Технологии:
- Angular
- TypeScript
- RxJS

## API:
Для получения данных о фильмах используется внешний API (TMDB - https://www.themoviedb.org/).
API-ключ хранится в environment файле и не добавлен в репозиторий.

## API Token:
Для работы приложения необходим API-токен.

1. Получите токен на сайте https://www.themoviedb.org/
2. Создайте файл `environment.ts` в папке `src/environments`
3. Добавьте в него токен.
